# CODINGTUBE DOCS

This project uses [Next.js](https://nextjs.org/) and [pnpm](https://pnpm.io/) for dependency management.

## Prerequisites
- Node.js >= 18
- pnpm installed globally (`npm install -g pnpm`)

## Installation

1. Install dependencies:
   ```bash
   pnpm install
   ```

2. Start the development server:
   ```bash
   pnpm dev
   ```

1. Static Build:
   ```bash
   pnpm build
   ```
