Para configurar alias en un proyecto de React con TypeScript y Vite, sigue estos pasos:

1. **Configura los alias en `tsconfig.json`:**

   Abre el archivo `tsconfig.json` en la raíz de tu proyecto y agrega la configuración de paths dentro del objeto `compilerOptions`. Aquí tienes un ejemplo:

   ```json
   {
     "compilerOptions": {
       "baseUrl": "./src",
       "paths": {
         "@components/*": ["components/*"],
         "@utils/*": ["utils/*"],
         "@styles/*": ["styles/*"]
       }
     }
   }
   ```

   En este ejemplo, estamos creando alias para las carpetas `components`, `utils` y `styles` dentro de la carpeta `src`.
