## Repaso de sintaxis de ciclo `for`

Si recordamos la estructura de un ciclo `for` para recorrer una lista es la siguiente:

```dart
List<String> nombres = ['Maria', 'Joaquin', 'Luisa'];

for (int i = 0; i <= nombres.length -1 ; i++) {
  // Hacer algo con el nombre
}
```	

Dentro del ciclo `for` tenemos que definir todo:


- `int i = 0` La inicializaciﾃｳn de la variable.
- `i <= nombres.length -1` La condiciﾃｳn que debe cumplir para que se ejecute el cﾃｳdigo.
- `i++` Cﾃｳmo va a ir cambiando la variable entre ciclo y ciclo.

Esto se puede simplificar muchﾃｭsimo, utilizando el ciclo `for in` y lo hacemos de la siguiente forma:

## El ciclo `for in` con `Listas`

```dart
List<String> nombres = ['Maria', 'Joaquin', 'Luisa'];

for (String nombre in nombres) {
  print('Hola ${nombre} 沁');
}
```	

1. Al inicio, utilizamos la palabra reservada `for` y entre parﾃｩntesis, declaramos la variable `nombre`. Como lo que vamos a recorrer es una lista de `Strings`, entonces esta variable tambiﾃｩn serﾃ｡ un `String`. 

2. Luego escribimos la palabra reservada `in` para especificar cual iterable vamos a recorrer, en este caso la lista `nombres`.

3. Despuﾃｩs, dentro de los parﾃｩntesis creamos el cﾃｳdigo que necesitemos, en nuestro ejemplo vamos a saludar a cada uno de las personas que aparecen en la lista.

El resultado es el siguiente:

```dart
Hola Maria 沁
Hola Joaquin 沁
Hola Luisa 沁
```

<Callout title="Nota">
En este ejemplo en particular, hemos creado la variable `nombre` porque va a representar cada uno de los nombres en la lista `nombres`, si fuesen `frutas`, escribirﾃｭamos `fruta`, o si deseas puede ser simplemente una `x`. Sin embargo recuerda que es recomendable ser lo mﾃ｡s descriptivos posibles con el nombre de las variables ya que esto ayuda a la legibilidad del cﾃｳdigo. 
</Callout>

## El ciclo `for in` con `Sets`

Ya hemos visto el uso de un `for in` en una lista ahora veamos su uso en un `Set`. Su uso es muy similar, ya que una lista y un set son estructuras muy parecidas: 

```dart
List<String> nombres = ['Maria', 'Joaquin', 'Luisa', 'Maria'];

Set<String> colores = {'rojo', 'azul', 'verde'};
```
<Callout title="Nota">
La diferencia es que un `Set` se crea con llaves y una lista con corchetes pero lo mﾃ｡s importante es que un `Set` no puede tener valores repetidos y una lista si. 
</Callout>

Entonces la forma en que se aplica el ciclo `for in` en un `Set` de la siguiente manera:

```dart
Set<String> colores = {'rojo', 'azul', 'verde'};

for (String color in colores) {
  print('El color es ${color}');
}
```	

Exactamente lo mismo a cﾃｳmo se aplica a una lista y el resultado es el siguiente:

```dart
El color es rojo
El color es azul
El color es verde
```
## El ciclo `for in` con `Mapas`

Por otro lado, para recorrer un mapa con `for in`, se necesita una lﾃｳgica extra, que la vamos a conocer a continuaciﾃｳn. 

Primero vamos a crear un mapa que muestra la edad de diferentes usuarios:

```dart
Map<String, int> edades = {
  'Maria': 20, 
  'Joaquin': 21, 
  'Luisa': 22
};
```	

En caso de que quieras profundizar en cﾃｳmo crear un mapa, te dejo este artﾃｭculo de [aquﾃｭ](https://codingtube.dev/docs/dart/maps-en-dart)

Ahora, para poder utilizar el ciclo `for in` con un mapa, es necesario tener en cuenta que a cada uno de sus elementos se los conoce como entradas. Entonces el recorrido lo vamos a hacer de entrada en entrada. 

Cada entrada es un par clave, valor y esto estﾃ｡ especificado en la definiciﾃｳn del mapa `Map<String, int> edades`. Es decir que aquﾃｭ estamos diciendo que cada entrada va a tener una clave que es de tipo String y su valor serﾃ｡ de tipo entero. 

Entonces, para recorrer un mapa con un for in, vamos a utilizar la palabra reservada `for` pero esta vez, dentro de los parﾃｩntesis, para declarar la variable que representarﾃ｡ cada una de las entradas, vamos a utilizar una clase de Dart que se llama `MapEntry`. 

<Callout title="Nota">
Ya vamos a conocer que son las clases en Dart en los artﾃｭculos posteriores. 
</Callout>

Lo ﾃｺnico que necesitas conocer de momento, es que esta clase `MapEntry` representa cada entrada de un mapa, por esto, tambiﾃｩn tenemos que especificar sus tipos, y cﾃｳmo cada una de las entradas de este mapa son `String` la clave e `int` el valor, entonces el tipo va a ser el mismo:

```dart
for(MapEntry<String, int> entrada in edades.entries) {
  // Hacer algo con la entrada
}
```	

Le vamos a poner el nombre de `entrada` a la variable y luego la palabra reservada `in` para indicar que vamos a recorrer las entradas del mapa `edades`, pero necesitamos especificar eso, asﾃｭ que vamos poner `edades` y luego `.entries`, que son sus entradas. 

Ya tenemos la estructura lista, ahora si podemos hacer algo con cada una de estas entradas como por ejemplo imprimir la siguiente frase:

```dart
for(MapEntry<String, int> entrada in edades.entries) {
  print('${entrada.key} tiene ${entrada.value} anﾌバs');
}
```	

Si ejecutamos el cﾃｳdigo vamos a ver lo siguiente.

```dart
Maria tiene 20 anﾌバs
Joaquin tiene 21 anﾌバs
Luisa tiene 22 anﾌバs
```
